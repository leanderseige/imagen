Welcome To The Imagen Project Page.

Generating Every Picture Representable By 65536 Bit.

Machine Will Tell You How Long It Takes Until
Your One Will Appear.

You Need A Java And JavaScript Capable Browser
Like Netscape Navigator Or Internet Explorer
To Visit This Site.

Errors May Occur If You Are Running MacOS. Your
Browser May Crash Because Of The Insufficient JVM.
Try The Internet Explorer For MacOS If You Are
Having Trouble Using Netscape.

Under Linux And MS Windows I Recommend The Latest
Netscape Navigator (4.7 Or Higher) .
